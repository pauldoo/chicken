# ZeroClaw Documentation & Source

This directory contains the ZeroClaw source repository checked out at the
version matching the installed binary. Use it to answer questions about
ZeroClaw configuration, features, channels, tools, and behaviour.

## Quick paths

- **Table of contents**:
  `repo/docs/book/src/SUMMARY.md`

- **Narrative docs (setup, channels, security, architecture, etc.)**:
  `repo/docs/book/src/`

- **CLI reference (generated from this binary)**:
  `repo/docs/book/src/reference/cli.md`

- **Config reference (generated from this binary)**:
  `repo/docs/book/src/reference/config.md`

- **Source code** (for deeper questions about behaviour):
  `repo/crates/` — all workspace crates
  `repo/src/` — main binary entrypoint and CLI commands
